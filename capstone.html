Project Description

For my capstone project, I created advanced AI software in Python. This software utilizes machine learning, which is a technology enabling computers to learn and make intelligent decisions from data patterns, without explicit programming. My project highlights the potential of AI to autonomously learn and adapt.

Goal

The goal of our project is to make an ai play connect 4, and improve through data of previous games. After this is successful, we want to be able to utilize this ai online, only using autonomous movements.

Steps

    ALPHA

  1. Make a workable connect 4 game in python.
  2. Make an ai that uses basic strategy.
  3. Be able to save game information to a csv file.
  4. Save 1 thousand unique game states to a csv file.
  5. Make different machine learning models, using the data created.
  6. Test accuracy of the models.

    7. BETA

  7. Read the website screen using a pillow.
  8. Convert the screen into a 2-d array containing the state of the game.
  9. Use mouse to input the move.

If I were to redo

Change it to chess - I was intimidated because I thought it would be too difficult. However, the only real changes would be creating the options for different moves.
